# Linked List

It's a singly linked list implementing in C

## Build and Run

1. Clone this project

```sh
$ git clone https://github.com/Yooootsuba/Linked-List.git
```

2. Change the directory

```sh
$ cd Linked-List
```

3. Build it

```sh
$ make
```

4. Run it

```sh
$ ./main.out
```
